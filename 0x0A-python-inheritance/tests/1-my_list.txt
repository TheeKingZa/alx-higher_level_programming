1-my_list.py:
    A module that defines the MyList class, which inherits from list and adds a method to print a sorted list.

MyList class:
    MyList inherits from the list class and provides additional functionality to print a sorted list.

Methods:
    - print_sorted(): Public instance method to print the list in ascending sorted order.

Test cases:

>>> in 1-my_list import MyList

# Create a MyList object and append some integers
>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(1)
>>> my_list.append(3)

# Test the print_sorted method
>>> my_list.print_sorted()
[1, 3, 5]

# Test with an empty list
>>> empty_list = MyList()
>>> empty_list.print_sorted()
[]

# Test with a list containing a single element
>>> single_element_list = MyList()
>>> single_element_list.append(42)
>>> single_element_list.print_sorted()
[42]

# Test with a larger list
>>> large_list = MyList([7, 2, 9, 4, 6])
>>> large_list.print_sorted()
[2, 4, 6, 7, 9]

# Test with negative integers
>>> negative_list = MyList([-3, -1, -5, -2])
>>> negative_list.print_sorted()
[-5, -3, -2, -1]

# Test with duplicate values
>>> duplicate_list = MyList([2, 2, 1, 1, 3, 3])
>>> duplicate_list.print_sorted()
[1, 1, 2, 2, 3, 3]
